# 🌌 Zenith Quest — Your Personal Sky Guide

**Discover what is happening above you tonight.**

A premium, hackathon-winning celestial exploration platform built with Next.js 16, TypeScript, Three.js, and Tailwind CSS.

## ✨ Features

- **Sky Score Engine** — Real-time 0-100 sky quality rating
- **Satellite Tracking** — Interactive Leaflet map with ISS, Starlink tracking
- **AI Sky Narrator** — OpenAI-powered sky summaries
- **3D Sky Dome** — Interactive Three.js celestial sphere
- **Missions & XP** — Gamified astronomy missions with badges
- **Celestial Timeline** — Tonight's events from sunset to sunrise
- **Analytics** — Recharts dashboards with observation trends
- **NASA Integration** — Live APOD, NEO, and Solar Activity

## 🚀 Getting Started

```bash
npm install
cp .env.example .env.local
npm run dev
```

The app runs fully in demo mode without any API keys.

## 🗂️ Tech Stack

Next.js 16 · TypeScript · Tailwind CSS v4 · Three.js · Framer Motion · Leaflet · Recharts · Zustand · OpenAI · NASA API · N2YO · Open-Meteo
